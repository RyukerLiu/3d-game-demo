# 最後一盞燈 / Last Light

A tiny browser-playable 3D survival-building story prototype. Find three materials before nightfall, build a shelter, and discover why the same shelter keeps appearing.

## Play

- **WASD / Arrow keys** — move
- **E** — build / interact
- Desktop browser only

## Stack

Three.js · TypeScript · Vite · GitHub Pages. All visuals are procedural geometry; no paid assets, APIs, backend, account, or database.

## Local development

```bash
npm install
npm run dev
npm run build
npm test
```

## Scope

One scene, one 60-second complete loop. No combat, inventory, crafting tree, physics engine, save, multiplayer, or character customization.
